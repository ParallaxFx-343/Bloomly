# Bloomly

Gratitude journaling app with a growing garden. Plant seeds of gratitude daily and watch your garden bloom.

## Features

- Daily gratitude entries with category selection
- Animated garden that grows with your entries
- 18 unique hand-crafted SVG plant illustrations (72 stages total)
- Dynamic sky backgrounds (morning, afternoon, evening, night)
- Ambient sounds and rain effects
- Streak tracking with milestone celebrations
- Dark mode support
- Premium tier with exclusive plants
- Rewarded ads to unlock daily premium plants
- Share your garden as a screenshot
- Fully offline — all data stored locally on device

## Tech Stack

- React Native (Expo SDK 55)
- TypeScript
- SQLite (local storage)
- react-native-reanimated (animations)
- react-native-svg (plant illustrations)
- RevenueCat (in-app purchases)
- Google AdMob (rewarded ads)

## Getting Started

```bash
npm install
npx expo start
```

## Privacy

All user data is stored locally on the device. No accounts required. See [Privacy Policy](privacy-policy.html).
